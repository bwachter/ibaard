char* cati ( char *  str,

Concatenate strings inline

Note
cati() uses a shared buffer. Don't use cati in functions which expect strings as arguments. Due to the shared memory bad things happen if the function gets called with cati in one argument.
Todo:
Add option to specify different levels of buffers to avoid the above problem

strthe first string
...the remaining strings, terminated by NULL
Returns
a pointer to the concatenated string
